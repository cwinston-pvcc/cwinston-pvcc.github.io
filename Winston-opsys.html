Operating Systems Terminology
TERM DEFINITION
Operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment
job queue jobs residing in mass storage waiting for execution
FIFO manner in which jobs are ordered - first-in, first-out
interactive processing programs which carry on a dialogue with the user while being executed
time-sharing an operating system design that provides service to multiple users at the same time
multiprogramming a time sharing technique in which time is divided into intervals and then the execution of each job is restricted to only one interval at a time
application software programs for performing tasks particular to a computer system's utilization
system software the operating system itself
utility software software units that extend or customize the capabilities of the operating system
graphical user interface a means by which objects to be manipulated, such as files and programs, are represented pictorially on a device's display as icons
shell older user interfaces, which communicated with users through textual messages using a keyboard and monitor screen
kernel the internal part of an operating system
device driver a component of the kernel which are the software units that communicate with the controllers to carry out operations on the machine's attached peripheral devices
file manager the software unit of the operating system that coordinates the use of the machine's mass storage facilities
memory manager the software component of the kernel that coordinates the machine's use of main memory
virtual memory the large "fictional" memory space created by paging
boot strapping the procedure that transfers the operating system from mass storage into main memory each time the machine is turned on
ROM read-only memory, the small portion of a computer's main memory where the CPU expects to find its initial program
deadlock the condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
spooling a technique in which a device driver holds data in mass storage for output at a later time
super user (or administrator) the person who gains highly privileged access to the operating system by identifying him- or herself as administrator curing the login procedure
multi-factor authentication a technique to secure computer accounts by requiring more than one piece of evidence to gain access
