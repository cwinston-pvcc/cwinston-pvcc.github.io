Networking & The Internet Terminology Table
TERM DEFINITION
Network Classifications
  • LAN
    		•  local area network
  • MAN
    		•  metropolitan area network
  • WAN
    		•  wide area network
    Network Models
  • bus network
    		•  LAN topology in which each machine is connected to a main cable called a bus
  • star network
    		•  LAN topology in which one machine serves as a central focal point to which all the others are connected
    Protocols
  • CSMA/CD Ethernet
    		•  Carrier Sense, Multiple Access with Collision Detection
  • CSMA/CA Wifi
    		•  Carrier Sense, Multiple Access with Collision Avoidance
    Combining Networks:
  • repeater
    		•  device that passes signals between two buses without considering their meaning
  • bridge
    		•  device that only passes a message when it is destined for a computer on the other side
  • switch
    		•  a bridge with multiple connections, allowing it to connect several buses rather than two
  • router
    		•  special purpose computers used for forwarding messages between networks
  • gateway
    		•  the point at which one network is linked to an internet, serving as a passageway between the network and the outside world
    Client/server model
  • P2P: peer-to-peer
    		•  a means of interprocess communication whereby processes provide service to and receive service from each other
  • distributed system
    		•  system that consist of software units that execute as processes on different computers
  • cloud computing
    		•  a distributed system whereby huge pools of shared computers on the network can be allocated for use by clients as needed
    Internet Addressing
  • Tier 1
    		•  Internet Service Providers that consist of very high-speed, high-capacity, international wide area networks
  • Tier 2
    		•  Internet Service Providers that are more regional in scope and less potent in their capabilities
  • Tier 3
    		•  an ISP operated by a single authority that is in the business of supplying Internet access to individual homes and businesses
    IP address
  • ICANN
    		•  Internet Corporation for Assigned Names and Numbers ICANN
  • DNS
    		•  domain name system; name servers that are used as an Internet-wide directory system
  • TLD: top-level domain
    		•  suffixes used to reflect a domain's classification
  • Country code TLDs
    		• two-letter TLDs for each country Country Code TLDs
    Internet protocols
  • FTP
    		•  File Transfer Protocol - a communications protocol used for the transfer of computer files from a server to a client on a network
  • HTTP
    		•  Hyper Text Transfer Protocol - a powerful communications protocol used for transmitting hypermedia documents, such as HTML
  • SMTP
    		•  Simple Mail Transfer Protocol - a communications protocol used to send and receive email
    World Wide Web
  • URL
    		•  a unique address used to locate and retrieve documents on the Web
  • client-side activities
    		•  everything in a web application that is displayed or takes place on the client end user device
  • server-side activities
    		•  everything that happens on the server, instead of on the client
    Internet four-layer software
  • application layer
    		•  the layer in the Internet communications software hierarchy that prepares the message and provides destination address and also receives the message
  • transport layer
    		•  the layer in the Internet communications software hierarchy that chops the message into packets at its origin and collects and reassembles them at its destination
  • network layer
    		•  the layer in the Internet communications software hierarchy that decides in which direction a packet should be sent at each step in its path through the Internet
  • link layer
    		•  the layer in the Internet communications software hierarchy that transfers the packet at message origin and receives it at the other end of the connection
    Internet is:
  • packet-switching network
    		•  a method of grouping data into packets that are transmitted over a digital network
  • fault tolerant
    		•  the capability of a system to continue operating without interruption when one or more of its components fail
  • redundant
    		•  the provision of duplicate, backup equipment or links that immediately take over the function of equipment or transmission lines that fail
    Cybersecurity
  • malware
    		•  collective name for malicious software
  • virus
    		•  software that infects a computer by inserting itself into programs that already reside in the machine
  • worm
    		•  autonomous software that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
  • Trojan horse
    		•  a program that enters a computer system disguised as a desirable program, such as a game or useful utility package
  • Spyware
    		•  malicious software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
  • phishing
    		•  a technique of obtaining information explicitly by simply asking for it
  • DDoS attack
    		•  distributed denial of service - the process of attacking a computer with the software being executed on a number of other host computers
    Protection
  • firewall
    		•  a network security system that monitors and controls traffic passing through a point in the network
  • spam filter
    		•  a firewall designed to block unwanted email
  • proxy server
    		•  a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server
  • antivirus software
    		•  software which is used to detect and remove the presence of known viruses and other infections
  • cryptography
    		•  the study of sending and receiving messages securely in the presence of adversaries
  • public key encryption
    		•  a two key technique by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure
